PID Controller Simulator

A **Proportional-Integral-Derivative (PID) controller** continuously calculates an error value as the difference between a **desired setpoint** and the **measured process variable**. The controller applies corrections using three terms:

  • Proportional (Kp): Adjusts the output **proportionally** to the current error.
  • Integral (Ki): Accounts for **past errors**, eliminating steady-state offset.
  • Derivative (Kd): Predicts future errors by analyzing **rate of change**.
